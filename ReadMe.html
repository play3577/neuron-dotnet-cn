NeuronDotNet

NeuronDotNet是一个开源工具，用于构建和运行基于“人工神经网络”的AI应用程序。 它是用C＃编写的，并与.NET 2.0平台兼容。 这个工具利用面向对象设计和模块化编程的潜力。 该项目的目标是支持人工智能应用程序的开发，并通过为不同类型的人工神经网络的创建，训练和使用提供集中的框架，支持实地研究。

发行说明

NeuronDotNet 3.0

发布日期：2008年8月20日

  • 支持具有任何非循环结构的层的神经网络
  • 支持One-One和Complete连接器
  • 支持反向传播网络和Kohonen SOM
  • 学习率使用可插拔函数（线性，对数和双曲线函数内置）将速率从其初始值更改为最终值。
  • 神经网络初始化模块是可插入的（随机，常数，NguyenWidrow和归一化随机函数内置）
  • 反向传播网络中使用的自定义激活功能是可插拔的（Sigmoid，正弦，tanh，对数和线性函数是内置的）
  • 对于Kohonen层，邻域函数是可插入的（高斯函数和墨西哥帽子函数是内置的）
  • 暴露了允许用户分析网络如何学习的各种事件
  • Kohonen层在形状上是平面的。 然而，我们可以具有圆形排和/或圆柱，使得它们获得圆柱形表面或环形表面的形状。
  • 支持六角形和矩形Kohonen格子拓扑
  • 培训集已定义为支持批处理培训
  • API添加自定义网络体系结构和学习算法
  • 层，连接器，网络和训练集实现ISerializable接口
NeuronDotNet 2.0

发布日期：2007年11月1日

  • 具有任何非循环结构的层的反向传播神经网络
  • 自定义激活功能是可插拔的
  • 增强的BackPropagation算法（使用Momentum项，重量衰减和抖动）。
  • 一层和层之间的完全连接
NeuronDotNet 1.0

发布日期：2007年5月3日

首次发布

  • 支持简单的前馈反向传播神经网络
  • 激活函数 - Sigmoid，Linear，Logarithminc，Sine或Tanh
